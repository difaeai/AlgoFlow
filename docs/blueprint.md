# **App Name**: AlgoFlow

## Core Features:

- Venue Connection: Connect to Exness or Binance accounts via API keys. API keys stored encrypted.
- Subscription Management: Allows users to subscribe to different tiers with varying monthly target ranges, handled via proof of payment.
- Automated Trading Bots: Orchestration of three bots (Analyzer, Strategist, Executor) to analyze data, create strategies, and execute trades within user-defined limits and tier constraints.
- Risk Management Controls: Configuration of risk settings such as stop loss percentage, daily drawdown limits, and maximum positions to manage trading risks. Paper trading mode is a tool to explore the system.
- Fee Management: Automated invoice generation for 3.5% company fee on profits when monthly targets are hit. Payments are handled off-platform with proof upload and admin approval.
- Referral Program: Track referrals within the app; allows admins to mark rewards as paid after verifying off-platform transfers.
- Admin Portal: Comprehensive admin portal for user management, payment proof review, fee management, and settings configuration.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) evoking trust and sophistication.
- Background color: Very light gray (#F0F0F0), almost white.
- Accent color: Electric purple (#BF00FF) for highlights, CTAs, and important alerts.
- Body text: 'Inter' (sans-serif) for a modern and readable interface.
- Headlines: 'Space Grotesk' (sans-serif) for headings and titles, especially numerical data summaries.
- Code Font: 'Source Code Pro' for display of API requests, code, or JSON data structures.
- Use financial-themed icons (charts, graphs, money) to represent data and actions. Line icons with consistent stroke width are preferred.
- Dashboard layout with clear separation of concerns - stats at the top, main content in the middle, secondary actions on the sides. Use of cards to group related information.
- Subtle animations for loading states and transitions to give a sense of responsiveness.
